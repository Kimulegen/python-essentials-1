1. **What is the output of the following snippet?**
    ```python
    lst = ["D", "F", "A", "Z"]
    lst.sort()

    print(lst)
    ```

    `['A', 'D', 'F', 'Z']`

2. **What is the output of the following snippet?**
    ```python
    a = 3
    b = 1
    c = 2

    lst = [a, c, b]
    lst.sort()

    print(lst)
    ```

    `[1, 2, 3]`

3. **What is the output of the following snippet?**
    ```python
    a = "A"
    b = "B"
    c = "C"
    d = " "

    lst = [a, b, c, d]
    lst.reverse()

    print(lst)
    ```

    `[' ', 'C', 'B', 'A']`
